# NinoLive

`NinoLive` is the public website project for the `NinoLive` domain.

## Current Goal

Ship a clean public landing page now, then evolve the site into a flexible home
for experiments, tools, and future projects.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS v4

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Phase 1 is designed for Vercel so the site can go live quickly with HTTPS and a
simple deployment flow.

The project structure is intentionally lightweight so it can move to the
existing Fasthosts VPS later with minimal rework.
